## 📖 故事背景

你在现实世界已经死亡。

但死亡并非终点 —— 你的祖宗在地下磕破了头为你换来了一场「**加时赛**」。代价很简单：你的灵魂被塞进了一具陌生的身体里。这具身体的原主人留下了一个意难平的执念。


**你有 7 天。**

扮演「宿主」，完成他未竟的心愿。成功了，送你重回人间。失败了……你的灵魂将永远迷失在虚空。

**简单来说这是一个魂穿并替原主完成愿望的故事。**

## 🎮 怎么玩

### 开局

进入游戏后，你需要完成一份「卖身契」：

- **你的名字**
- **你的性别** — 你希望宿主的性别是什么
- **世界背景** — 选择你即将前往的世界：中世纪？赛博朋克？古代权谋？武侠修仙？随你决定
- **额外设定** — 对这个世界的任何愿望

填完后按下「**RUNNN!!!!**」，系统会为你生成一个独一无二的世界。

### 核心机制

| 机制 | 说明 |
|:---|:---|
| ⏳ **7 日倒计时** | 你只有 7 天完成宿主的执念，逾期即死 |
| 🔴 **七宗罪日课** | 每一天对应七宗罪之一，你必须在当日完成对应的每日任务，否则**当场抹杀** |
| 👁️ **规则怪谈** | 每个世界都有诡异的规则。不遵守？你可以试试看 |
| ❤️ **HP / SAN** | 生命值归零你就死了；理智值归零……会发生更可怕的事 |
| 💰 **信用点** | 可以在补给商店购买道具续命 |

### 界面指南

游戏界面底部有一排功能模块，点击图标可以打开：

- 🧑 **角色档案** — 查看你的宿主身份、当前状态
- 🗺️ **行动日志** — 主线任务和每日任务追踪
- ⚠️ **生存协议** — 当前世界的规则怪谈，**务必认真阅读**
- 🛒 **补给商店** — 用信用点购买续命道具
- 📦 **随身背包** — 管理你的物品，可以在对话中使用
- 👥 **社会羁绊** — 查看 NPC 好感度和互动记录

### 游戏提示

> 💡 **你的每一句话都会影响世界走向。** AI 会根据你的行为实时调整剧情、NPC 态度、任务进度。没有固定路线，没有标准答案。

> ⚠️ **规则怪谈不是摆设。** 违反规则的后果是真实的，可能是掉 SAN、掉 HP，也可能是即死。

> 🎭 **尝试不同的行动风格。** 系统会为你生成多种行动建议：理性分析、大胆试探、暴力突破、讽刺调侃……选择你的方式，或者自己输入任何你想做的事。
